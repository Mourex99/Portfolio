Sobre

Estou inserido na área de TI, com foco em programação e infraestrutura, sempre buscando aprender e aplicar novas tecnologias. Atualmente, estou aprimorando meus conhecimentos em Linux, embora o Windows ainda seja meu principal sistema para codificação.

Tenho sólida experiência em administração de servidores (Windows Server e Ubuntu Server), além de trabalhar com virtualização através do Hyper-V. Essa experiência me levou a explorar cloud computing, onde venho me aprofundando em Amazon Web Services (AWS), utilizando serviços como EC2 para criação e gerenciamento de máquinas virtuais, S3 para armazenamento seguro e escalável, Elastic Beanstalk para implantação de aplicações, ECS e Lambda para gerenciamento de contêineres e funções serverless, além de bancos de dados com Amazon RDS e DynamoDB.

Na programação, destaco meus conhecimentos em Python, com foco no desenvolvimento web utilizando Flask para criação de APIs REST e integração com bancos de dados no qual tenho conhecimento em MySQL, SQLite e breves experiências com PostgreSQL e MariaDB. Também possuo experiência com HTML, CSS, JavaScript.

Minha paixão por tecnologia e aprendizado contínuo me mantém atualizado com as principais tendências, permitindo que eu ofereça soluções inovadoras e otimizadas, especialmente nas áreas de desenvolvimento e cloud computing. Meu objetivo é contribuir para o sucesso e inovação das organizações com as quais trabalho.

Focos

Atualmente, meu foco é crescer na área de programação e infraestrutura, com ênfase em soluções voltadas para a nuvem. Para alcançar esse objetivo, estou estudando profundamente as plataformas AWS e Azure, que são essenciais para a gestão eficiente de infraestrutura em nuvem e a implementação de arquiteturas escaláveis e seguras.

Além de meu interesse em cloud computing, tenho um sólido conhecimento em programação. Trabalho com Python, uma linguagem que utilizo para automação e desenvolvimento de soluções customizadas. Minha experiência também inclui Java, HTML, CSS e JavaScript, ferramentas que me permitem criar aplicações web robustas e interativas assim como está na qual está visualizando no momento.

Tenho experiência prática na administração de servidores, tanto com Windows Server quanto com Linux Server. Essa experiência inclui a configuração, manutenção e otimização de servidores, o que é crucial para garantir a estabilidade e o desempenho das infraestruturas de TI.

Essa combinação de habilidades em programação, cloud computing e redes me proporciona uma visão integrada e estratégica para enfrentar desafios complexos e contribuir significativamente para a inovação e eficiência das operações de TI. Estou comprometido com o aprendizado contínuo e a aplicação das melhores práticas para alcançar resultados excepcionais.
